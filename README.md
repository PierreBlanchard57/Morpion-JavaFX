#  Projet : Morpion JavaFX

Morpion conçu avec JavaFX

---

## 📝 Présentation

Ce projet est un projet d'etudes dont l'objectif était de concevoir un morpion en utilisant JavaFX.

### Stack Technique
* **Langage** : Java
* **Framework/Lib** : JavaFX
* **Outils** : IntelliJ

---

## ⚙️ Installation

Suivez ces étapes pour éxécuter le projet.

### Prérequis
* **Systeme d'exploitation** : Windows 10,Windows 11
* **Logiciels** : Android studio,IntelliJ ou autre IDE Java

### Étapes d'installation

1. **Clonez le dépôt.**
2. **Ouvrez votre IDE et selectionnez le projet.**
3. **Executez le projet.**
